# macOS Terminal Commands

This repository contains terminal commands I practiced for moving through folders, creating files, renaming items, and organizing a project from the command line.

## Navigation Commands

| Command          | What It Does                                     |
| ---------------- | ------------------------------------------------ |
| `pwd`            | Shows the current folder location                |
| `ls`             | Lists files and folders in the current directory |
| `cd folder-name` | Moves into a folder                              |
| `cd ..`          | Moves back one folder                            |
| `clear`          | Clears the terminal screen                       |

---

## File & Folder Commands

| Command                    | What It Does                              |
| -------------------------- | ----------------------------------------- |
| `mkdir folder-name`        | Creates a new folder                      |
| `touch file-name`          | Creates a new file                        |
| `touch README.md`          | Creates a README file                     |
| `mv old-name new-name`     | Renames a file or folder                  |
| `mv file-name folder-name` | Moves a file into another folder          |
| `rm file-name`             | Deletes a file                            |
| `rm -r folder-name`        | Deletes a folder and everything inside it |
